Regras do jogo:

O jogo [inserirNome] é um jogo de aventura com puzzles. Existem 3 fases e em cada uma uma chave para a fase seguinte.

  • Para se movimentar:

    • W: Movimenta uma unidade para cima;
    • A: Movimenta uma unidade para esquerda;
    • S: Movimenta uma unidade para baixo;
    • D: Movimenta uma unidade para direita;
    • I: Interage com o objeto que está em cima.

  • Elementos das fases:

    • &: Jogador.
    • *: Parede, o jogador ao se movimentar não pode passar pela parede.
    • @: Chave para abrir a porta para finalizar a fase, a porta abre no momento que o jogador interage com a chave.
    • D: Porta fechada.
    • =: Porta aberta quando o jogador interage com a chave. Em que mapas que tenham mais de uma porta, deve-se interagir com a última porta destrancada para passar de fase
    • O: Botão que o usuário pode interagir, o botão fica no chão e o jogador deve ficar em cima dele para poder interagir.
    • #:Um espinho. Quando o jogador toca no espinho ele perde uma vida, caso ele morra três vezes, o jogo volta para o menu principal.
    • (_): Símbolo que representa um teletransporte. O teletransporte sempre vem em pares, quando o jogador toca em um ele é transportado para o outro e vice-versa.
    • {, }: Símbolos que representam paredes que só podem ser atravessadas por um lado. A parte aberta da chave é por onde se passa.
    • v, >, <, ^: Símbolos que representam uma flecha. Caso o jogador encoste numa, a fase reinicia e ele perde uma vida.
